Plot_1 Seattle Weather

Plot Interpretation

In this plot, I adjusted the size of the visualization, added the color to the scatter plot, and changed the font size of the axis labels. This plot shows the highest temperature in each day and the actual weather of that day. The x-axis is representing the time in Month and Year. each unit appears to be quarterly. As we can see when the temperature is high the most likely weather is going to be sunny. When it rains the temperature rarely exceeds 25 degrees celsius. As we can see from this graph Seattle rains a lot. The heavy raining season occurs across the end of the summer of the current year to next year's beginning of the summer season.

Code Example 


							var spec = {
								"$schema": "https://vega.github.io/schema/vega-lite/v5.json",
								"width": 800,
								"height": 400,
								"data": {
									"url": "https://vega.github.io/vega-datasets/data/seattle-weather.csv"
								},
								"mark": "point",
								"encoding": {
									"x": {
										"type": "temporal", 
										"field": "date",
										"axis": {
											"titleFontSize": 15
										}
									},
									"y": {
										"field": "temp_max",
										"type": "quantitative",
										"axis": {
											"titleFontSize": 15
										}
									},
									"color": {
										"field": "weather",
										"type": "nominal"
									}
								}
							};

Plot_2 Disasters & Death

Plot Interpretation

In this plot, I added the facts feature which shows the death counts in each year and relates to each disaster that occurred in that year. As we can see from the bar plot. a natural disaster like Flood kills a lot of people, then the Drought seems to kill many people, which the death toll is right under Flood. Then we can see in between 1905 to 1925 a lot of people are dead due to the Epidemic. So I looked up online I find that during that period diseases like Influenza(Spanish Flu) outbreak (1918) killed a lot of people.
