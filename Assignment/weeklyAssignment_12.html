Plot_1 Seattle Weather

Plot Interpretation

This plot shows the highest tempurture in each daay and the actual weather of that day. X-axis is representing the time in Month and Year. each unit appers to be quarterly.

Code Example 


							var spec = {
								"$schema": "https://vega.github.io/schema/vega-lite/v5.json",
								"width": 800,
								"height": 400,
								"data": {
									"url": "https://vega.github.io/vega-datasets/data/seattle-weather.csv"
								},
								"mark": "point",
								"encoding": {
									"x": {
										"type": "temporal", 
										"field": "date",
										"axis": {
											"titleFontSize": 15
										}
									},
									"y": {
										"field": "temp_max",
										"type": "quantitative",
										"axis": {
											"titleFontSize": 15
										}
									},
									"color": {
										"field": "weather",
										"type": "nominal"
									}
								}
							};

Plot_2 Disasters & Death

Plot Interpretation

This plot shows the the death counts in each year, and relates to each disaster that occured in that year. As we can see from the bar plot. natural disaster like Flood kills a lot of people, then the Drought seems to kill many people, which the death toll is right under Flood. Then we can see in 1905 to 1925 a lot people are dead due to Epidemic.

Code Example 


							var spec = {
								"$schema": "https://vega.github.io/schema/vega-lite/v5.json",
								"width": 800,
								"height": 400,
								"data": {
									"url": "https://vega.github.io/vega-datasets/data/disasters.csv"
								},
								"mark": "bar",
								"encoding": {
									"x": {
										"type": "temporal", 
										"field": "Year",
										"axis": {
											"titleFontSize": 15
										}
									},
									"y": {
										"field": "Deaths",
										"type": "quantitative",
										"axis": {
											"titleFontSize": 15
										}
									},
									"color": {
										"field": "Entity",
										"type": "nominal"
									}
								}
							};
